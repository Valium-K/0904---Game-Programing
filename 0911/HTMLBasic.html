HTML

HTMLHyper Markup Language은 W3C에서 보급하는 웹을 통해 볼 수 있는 문서를 만들때 사용하는마크업Markup언어의 한 종류이다. 특히 하이퍼텍스트를 작성하기 위해 개발되었으며, 인터넷에서 웹을 통해 접근되는 대부분의 웹 페이지들은 HTML로 작성된다.

XML

인터넷 웹 페이지를 만드는HTMLSGML을 개선한 마크업 언어이다. 데이터의 저장 밑 변환, 등이 향상되었고, 웹 페이지의 관리가 효율적이다.
